# 趣学伴 1.6.1 发布说明

更新时间：2026-04-27

## 版本定位

`1.6.1 = UI 系统统一版`

本版本是 1.6 后的小版本修正，重点统一家长端 UI，不修改数据库结构。

## 已完成范围

1. 新增 `PageToolbarTitle`、`FilterBar`、`EmptyPanel` 三个轻量 UI 组件。
2. 统一能力模型、任务管理、学习计划、阅读中心、图书馆、学习报告、统计分析、成就管理、设置的页面顶栏。
3. 学习计划和任务管理的主筛选区统一为标准 `FilterBar`。
4. 阅读中心、任务管理、成就管理空状态统一为 `EmptyPanel`。
5. 成就管理卡片从旧的大圆角重阴影样式收敛为普通卡片样式。
6. 任务详情页接入标准顶栏，主要卡片和危险确认弹窗收敛为统一样式。
7. 图书详情页外层、主要操作按钮、阅读状态卡片和主要弹窗收敛为统一样式。
8. 阅读中心和成就管理的确认弹窗收敛为统一弹窗样式。
9. 版本号更新为 `1.6.1`，后端 `/api/version` 同步返回 `1.6.1`。
10. 修正 `pnpm-workspace.yaml`，纳入 `frontend` 和 `backend`，避免服务器部署时 pnpm 跳过前后端依赖。

## 数据库说明

本版本未修改 Prisma schema，未新增 migration，不需要重置数据库，不需要 seed。

## 部署提醒

服务器更新后建议使用 pnpm 重新安装：

```bash
cd /home/ubuntu
pnpm install --frozen-lockfile --prod=false
```

然后分别构建前后端，并将前端 `dist` 同步到 Nginx 静态目录。
