# 趣学伴 1.2 发布说明

更新时间：2026-04-25

## 版本定位

`1.2` 是在 `1.0 / 1.0.1` 稳定基础上的一次功能确认与结构收口版本。

目标：

- 将已经验证通过的修改正式整理入主线
- 更新到正式环境
- 形成新的稳定版本基线
- 为 `1.5` 的体验重构与结构优化做准备

---

## 1.2 计划纳入的内容

### 一、核心功能与流程修复

- 任务详情编辑后，任务列表自动刷新
- 孩子端登录概念移除后，孩子新增/编辑不再依赖 PIN/密码
- 当前孩子上下文在更多页面中统一生效
- 学习统计页去掉重复孩子筛选和低价值提示块
- 学习报告页去掉重复孩子筛选，统一跟随左侧当前孩子

### 二、公共框架与页面统一第一轮

- 左侧导航与品牌区统一
- 页面命名统一
  - `概览` -> `首页`
  - `数据统计` -> `学习统计`
  - `AI 报告` -> `学习报告`
- Button / Card / Badge / Skeleton 第一轮统一
- 去掉页面中无业务意义的结构标签
  - `BRAND`
  - `ACCOUNT`
  - `分析与复盘`

### 三、阅读体系第一轮重构

- 图书馆首页进入“书架化”方向
- 图书卡片瘦身
- 图书详情页两轮收口
- 阅读页开始向“阅读中心”靠拢

### 四、设置页统一第一轮

- 家庭设置统一
- 账户信息统一
- 孩子管理统一
- 学习设置与通知偏好暂不放入 1.2，后续随设置体系二次收口再恢复

---

## 1.2 不包含的内容

以下内容不放入 `1.2`，统一进入 `1.5` 或专项：

- 图书馆完整重构
- 阅读页完整重构
- 学习统计完整重构
- 学习报告完整重构
- 设置页全部子页完整统一
- 学习设置与通知偏好恢复
- 性能与加载体验专项优化
- 图书封面与历史导入专项
- 能力模型与 AI 升级能力
