# 趣学伴 1.6.1 UI 统一规范

更新时间：2026-04-27

## 版本定位

`1.6.1 = UI 系统统一版`

本版本不新增业务能力，不修改数据库结构，重点统一家长端页面骨架、按钮、卡片、筛选区和空状态。

## 页面骨架

家长端一级页面统一使用：

1. `PageToolbar`
2. 核心指标或筛选区
3. 主内容区
4. 辅助信息、空状态或弹窗

页面容器统一：

```tsx
<div className="mx-auto max-w-[1360px] space-y-5">
```

## 顶栏

标准顶栏使用 `PageToolbarTitle`：

- 图标：`h-10 w-10 rounded-lg bg-primary/10 text-primary`
- 标题：`text-base font-semibold`
- 描述：`text-xs sm:text-sm text-slate-500`
- 右侧按钮：`h-11 rounded-xl`

## 按钮

按钮分为五类：

- 主操作：新增、编辑、发布，使用紫蓝渐变。
- 次操作：同步、推送、分析，使用蓝色。
- 导出：导出报告、数据或图片，使用绿色。
- 普通操作：筛选、日期、返回，使用白底描边。
- 危险操作：删除、重置，使用红色。

尺寸：

- 主按钮：`h-11 rounded-xl`
- 筛选按钮：`h-10 rounded-lg`
- 图标按钮：`h-10 w-10 rounded-lg`

## 卡片

普通页面卡片：

```tsx
rounded-xl border border-slate-200 bg-white shadow-sm
```

重点卡片可以使用浅色背景或少量渐变。`rounded-3xl` 和 `shadow-2xl` 只用于弹窗或特殊强调，不用于普通页面内容。

## 筛选区

页面主维度筛选使用 `FilterBar`。

时间、日期、导出、新建等操作放在 `PageToolbar.right`。分类、状态、分组等主内容筛选放在顶栏下方。

## 空状态

统一使用 `EmptyPanel`：

- 图标
- 标题
- 一句说明
- 可选主操作按钮

文案需要明确下一步，不使用过长解释。
