# 趣学伴 1.6 发布说明

更新时间：2026-04-27

## 版本定位

`1.6 = 业务逻辑梳理版`

本版本不做数据库结构升级，重点是把家长端的核心业务对象关系先理顺：

- 能力模型用于定义标准和发现短板。
- 目标用于承接能力点并形成阶段方向。
- 任务用于承接目标并落到可执行动作。
- 学习计划用于把任务排到具体日期。
- 今日概览用于当天执行和打卡。
- 报告和成就后续从这些行为数据中汇总。

## 已完成范围

1. 新增家长端 `能力模型` 页面。
2. 能力模型先按 L1-L5 对应一年级到五年级展示。
3. 能力模型静态包含四类能力：学科能力、思维与认知、学习习惯、体育与健康。
4. 目标页增加能力模型承接口径，展示能力点、关联任务、复盘节奏和阶段进度。
5. 任务页新增 `1.6 业务关联` 字段：任务类型、年级等级、能力分类、能力点、关联目标。
6. 学习计划页在计划发布、移动、删除后同步刷新今日概览相关数据。
7. 今日概览顶部移除重复的 `安排计划` 入口。
8. 仪表盘时间筛选区右对齐，保持页面顶栏一致性。
9. 新增 `docs/ROADMAP_1_6.md`，记录 1.6 业务逻辑和后续范围边界。

## 数据库说明

本版本未修改 `backend/prisma/schema.prisma`，未新增 Prisma migration。

正式环境上线时不需要重置数据库，也不需要执行 seed。正常拉代码、安装依赖、构建和重启服务不会覆盖正式数据。

禁止在正式环境执行：

```bash
pnpm prisma:reset
npx prisma migrate reset
prisma migrate reset
pnpm prisma:seed
npx prisma db push --force-reset
```
